# 🍽 Microwave Oven Simulation (Embedded C)

## 📌 Overview
This project simulates the working of a microwave oven using Embedded C. It was developed during my internship at Emertxe as part of hands-on embedded systems training.

The system mimics real microwave functionalities such as:
- Timer setting
- Cooking modes
- Start / Stop operation
- Buzzer indication
- Door open/close detection
